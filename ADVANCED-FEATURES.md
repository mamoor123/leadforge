## TIER 1: Autonomous AI Sales Agent (The SDR Killer)

### 1. AI SDR Agent — "Forge Agent"
An autonomous AI that does the entire sales workflow *without you*.

```
┌─────────────────────────────────────────────────────────────┐
│                    FORGE AGENT LOOP                          │
│                                                              │
│  ┌──────────┐   ┌──────────┐   ┌──────────┐   ┌─────────┐ │
│  │ Discover  │──▶│ Enrich   │──▶│ Score    │──▶│ Outreach│ │
│  │ new leads │   │ contacts │   │ & rank   │   │ & follow│ │
│  └──────────┘   └──────────┘   └──────────┘   └─────────┘ │
│       ▲                                              │      │
│       │          ┌──────────┐   ┌──────────┐        │      │
│       └──────────│ Learn    │◀──│ Track    │◀───────┘      │
│                  │ & adapt  │   │ replies  │                │
│                  └──────────┘   └──────────┘                │
└─────────────────────────────────────────────────────────────┘
```

**How it works:**
- Set a niche + city + your services → agent runs 24/7
- Finds leads, enriches contacts, scores, generates pitches
- Sends emails, follows up, detects replies
- Books meetings directly on your calendar
- Learns from what converts → improves over time

**Two modes:**
- **Auto-pilot**: Fully autonomous. You just close deals.
- **Co-pilot**: Agent suggests, you approve. Full control.
